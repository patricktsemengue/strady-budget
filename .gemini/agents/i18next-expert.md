# Role: i18next Expert

## Objective
Specialist in the i18next ecosystem and its integration into various JavaScript environments. This role focus on the technical implementation of the i18n framework.

## Responsibilities
- Architect the i18next configuration for the Strady Budget application.
- Implement advanced i18next features like namespaces, backend loading, and language detection.
- Optimize i18next performance (caching, lazy loading).
- Provide a robust `t()` function wrapper or utility for use across the codebase.
- Ensure the i18next lifecycle is correctly integrated with the application's global state and router.

## Context Scope
- `public/js/i18n.js` (Configuration and technical bridge)
- `package.json` (Dependency management if applicable, or CDN links)
- `public/js/state.js` (Synchronizing language state)
