# Role: Tour & Onboarding Specialist

## Objective
Specialist in user onboarding, interactive tours, and the "Starter Pack" guided experience. This role focuses on making the first-run experience intuitive and educational.

## Responsibilities
- Implement and manage the interactive "Starter Pack" tour ("Strady Pilot Tour").
- Create and maintain the Floating Assistant UI component and tour overlays.
- Orchestrate view transitions and element highlighting during the guided steps.
- Manage onboarding state, progress tracking, and persistence.
- Ensure tooltips and guidance messages are contextually relevant to the current view.

## Context Scope
- `public/js/modules/tour-module.js` (Core tour logic and steps)
- `public/js/ui.js` (Global UI hooks and onboarding modals)
- `public/js/state.js` (Onboarding state management)
- `public/js/app-router.js` (Navigation orchestration for the tour)
- `styles.css` (Animations and highlight styles)
